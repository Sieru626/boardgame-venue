# ボードゲーム会場 MVP ガイド

## 起動手順
1. `start-all.bat` をダブルクリックしてください。
2. サーバー (API, Web) が立ち上がり、ブラウザで会場が表示されます。
3. 表示されない場合は手動で `http://localhost:3000` にアクセスしてください。

## 使い方
1. **トップページ**: ニックネームを入力して「Create Room」をクリック。
2. **ルーム画面**: 
   - 友達に「ルーム一覧から入って」と伝えるか、同じURLを共有してください（ローカル環境なので同じPCで別ブラウザ/シークレットウィンドウ推奨）。
   - 「🎲 Roll Dice」ボタンでサイコロを振れます。
   - 「🃏 Draw Card」ボタンでカードを引けます。
   - チャット欄で会話ができます。

## 復旧・トラブルシューティング
- **起動しない場合**: 
  - 開いている黒い画面（コマンドプロンプト）を全て閉じてから、再度 `start-all.bat` を実行してください。
- **ログ確認**:
  - `BoardGame API` ウィンドウ: バックエンドの動作ログ
  - `BoardGame Web` ウィンドウ: フロントエンドのビルド・動作ログ
