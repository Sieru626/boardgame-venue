# Cursor で「全部できる」権限の設定（日本語UI）

## 1. すでに作成済みのファイル

次のファイルはすでに作成してあります（変更不要です）。

- **場所:** `C:\Users\user\.cursor\sandbox.json`
- **内容:** サンドボックス無効 ＋ ネットワーク許可

---

## 2. Cursor の画面から行う設定

### 手順A: 設定画面を開く

1. Cursor のメニューで **「ファイル」** をクリック
2. **「ユーザー設定」** または **「設定」** をクリック
3. 一覧から **「Cursor 設定」**（または「Cursor Settings」）を選ぶ  

   **別の方法:** キーボードで **Ctrl + ,** を押して設定を開き、検索欄に **「エージェント」** または **「Auto-Run」** と入力する

---

### 手順B: エージェントの自動実行を「全部許可」にする

設定画面で次を探して変更します。

| 項目（日本語で表示されていそうな名前） | 選ぶ値 |
|----------------------------------------|--------|
| **自動実行モード** / **Auto-Run モード** / **エージェントの実行モード** | **「すべて実行」** または **「Run Everything」** |
| **自動実行時のネットワーク** / **ネットワークアクセス** | **「すべて許可」** または **「Allow All」** |

- 「サンドボックスで実行」「毎回確認」などは選ばない
- 「すべて実行」「すべて許可」にすると、コマンド実行やネットワークが許可され、以前のように動きやすくなります

---

### 手順C: 反映のために Cursor を再起動

1. Cursor を終了する（ウィンドウを閉じる）
2. もう一度 Cursor を起動する

これで `sandbox.json` と画面の設定が反映されます。

---

## 3. 設定画面の探し方（検索で見つける場合）

設定の検索ボックスに次の語を入れて、該当項目を開きます。

- **エージェント** … エージェント関連の設定
- **自動実行** … Auto-Run の項目
- **サンドボックス** … サンドボックス・ネットワークの項目
- **Run Everything** … 英語のまま表示されている場合

「すべて実行」「すべて許可」に変更し、Cursor を再起動すれば設定完了です。
